Sniffles

Sniffles is a structural variation caller using third generation sequencing (PacBio or Oxford Nanopore). It detects all types (deletions, duplications, insertions, inversions and translocations) of structural variations using evidence from split-read alignments, high-mismatch regions, and coverage analysis. Please note the current version of Sniffles requires output from BWA-MEM with the optional SAM attributes enabled! Sniffles is currenlty used in the Genome in the Bottle project as well as various human cancer projects.
